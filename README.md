# Proiect cu 3 modele de machine learning:

### 1. Predictia pretului Bitcoin
- Regresie liniara
- Regresie multiliniara
- Regresie polinomiala

### 2. Prezicerea longevitatii carierei NBA
- Regresie logistica
- Clasificare binara

### 3. Recunoasterea cifrelor scrise de mana
- Clasificare multiclasa: KNN, Nearest centroid, Arbori decizionali
- MDS
- Clustering: K-means, K-medoids, Clustering ierarhic
